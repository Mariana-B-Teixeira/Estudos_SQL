# 📘 Exercícios SQL - Curso em Vídeo

Este diretório reúne os **exercícios de SQL** realizados durante o estudo com o **Curso em Vídeo**.  
Ele organiza os treinos em subpastas, cada uma com exemplos de bancos de dados e scripts para praticar comandos fundamentais.

---

## 🎯 Objetivo

- Consolidar os conceitos básicos de SQL.  
- Praticar a criação e manipulação de bancos de dados.  
- Fixar comandos de **DDL** (Data Definition Language), **DML** (Data Manipulation Language) e consultas simples.  

---

## 📂 Estrutura das Pastas

- **Treino DDL e DML_Exercícios** → Exercícios focados em criação de tabelas, inserção, atualização e exclusão de registros.  
- **Treino básico SQL_Exemplo_Banco_de_Dados** → Banco de dados fictício usado como exemplo para treinar consultas básicas e manipulação de dados.  

---

## 🛠️ Conteúdos Praticados

- `CREATE DATABASE` e `CREATE TABLE` → Estruturação de bancos e tabelas.  
- `INSERT INTO`, `UPDATE`, `DELETE` → Manipulação de registros.  
- `SELECT`, `WHERE`, `ORDER BY` → Consultas básicas com filtros e ordenação.  

---

## 🚀 Como Executar

1. Abra seu ambiente de banco de dados (MySQL, MariaDB ou outro compatível).  
2. Escolha uma das subpastas.  
3. Copie os comandos SQL dos arquivos correspondentes.  
4. Execute-os passo a passo para observar os resultados.  
5. Teste variações das consultas para reforçar o aprendizado.  

---

## 📚 Aprendizados

- [x] Diferença entre DDL e DML.  
- [x] Criação e manipulação de bancos de dados fictícios.  
- [x] Consultas simples para explorar dados.
