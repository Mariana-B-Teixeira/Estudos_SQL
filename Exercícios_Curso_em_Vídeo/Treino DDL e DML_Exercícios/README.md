# 📘 Treino DDL e DML - Exercícios

Este diretório reúne os exercícios práticos de **DDL (Data Definition Language)** e **DML (Data Manipulation Language)** realizados como parte do estudo de SQL no Curso em Vídeo.

---

## 🎯 Objetivo

- Consolidar os conceitos fundamentais de **definição de estruturas** (DDL) e **manipulação de dados** (DML).
- Praticar comandos essenciais para criar, alterar e excluir tabelas, além de inserir, atualizar e remover registros.

---

## 🛠️ Conteúdos Praticados

### 🔹 DDL (Data Definition Language)
- `CREATE DATABASE` → Criação de banco de dados.
- `CREATE TABLE` → Definição de tabelas e colunas.
- `ALTER TABLE` → Alteração da estrutura de tabelas.
- `DROP TABLE` → Exclusão de tabelas.

### 🔹 DML (Data Manipulation Language)
- `INSERT INTO` → Inserção de registros.
- `UPDATE` → Atualização de dados existentes.
- `DELETE` → Remoção de registros.
- `SELECT` → Consultas básicas e avançadas.
