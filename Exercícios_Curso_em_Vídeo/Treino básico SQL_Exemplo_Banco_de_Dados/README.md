# 🎶 Ariana Grande - Avaliações

Este diretório contém um **exemplo de banco de dados criado para treinar SQL**, usando como tema os álbuns da cantora e compositora **Ariana Grande**.  
A ideia é tornar o aprendizado mais envolvente, simulando um sistema de avaliações musicais.

---

## 🎯 Objetivo

- Praticar comandos básicos e intermediários de SQL.  
- Estruturar tabelas relacionadas a **álbuns, músicas e avaliações**.  
- Fixar conceitos de **DDL** (definição de tabelas) e **DML** (manipulação de dados).  
- Simular um cenário realista de aplicação de SQL em um banco de dados temático.  

---

## 🗄️ Sobre o Banco de Dados

O banco de dados foi projetado para representar:  

- **Tabela de Álbuns** → informações como título, ano de lançamento e gravadora.  
- **Tabela de Músicas** → faixas pertencentes a cada álbum, com duração e ordem.  
- **Tabela de Avaliações** → notas e comentários atribuídos aos álbuns.  

📌 **Como funciona:**  
- Cada álbum pode ter várias músicas associadas.  
- As avaliações permitem analisar e comparar diferentes fases da carreira da artista.  
- Os dados são fictícios, criados apenas para fins didáticos, mas seguem a lógica de um sistema real.  

---

## 🛠️ Conteúdos Praticados

- `CREATE DATABASE` → Criação do banco de dados de exemplo.  
- `CREATE TABLE` → Estruturação das tabelas de álbuns, músicas e avaliações.  
- `INSERT INTO` → Inserção de dados simulados.  
- `SELECT` → Consultas para explorar os álbuns e notas.  
- `JOIN` → Relacionamento entre tabelas (álbuns ↔ músicas ↔ avaliações).  
- `UPDATE` e `DELETE` → Manipulação dos registros.  

---

## 🚀 Como Executar

1. Abra seu ambiente de banco de dados (MySQL, MariaDB ou outro compatível).  
2. Copie os comandos SQL dos arquivos desta pasta.  
3. Execute-os para criar o banco de dados de exemplo.  
4. Teste consultas e manipulações para observar os resultados.
